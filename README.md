# PickUpLine

A small React + Vite app that fetches a random pickup line at the click of a
button, with copy and share actions.

## Running locally

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — start the dev server
- `npm run build` — build for production
- `npm run lint` — run ESLint
- `npm run preview` — preview the production build locally

## Configuration

The pickup line API URL is read from `VITE_PICKUP_LINE_API_URL` in `.env`.
